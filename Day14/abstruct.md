7. データベースの「結合 (JOIN)」操作
リレーショナルデータベースで共通列を使ってテーブルを組み合わせる操作を、データ構造で表現しています。

Java (Stream API などを使ったメモリ上の内部結合)

Java
import java.util.List;

record User(int id, String name) {}
record Order(int orderId, int userId, String item) {}
record UserOrderCombined(String userName, String item) {} // 結合結果

public class JoinExample {
    public List<UserOrderCombined> innerJoin(List<User> users, List<Order> orders) {
        // userId が一致する要素同士を組み合わせて結合する
        return orders.stream()
            .flatMap(order -> users.stream()
                .filter(user -> user.id() == order.userId())
                .map(user -> new UserOrderCombined(user.name(), order.item())))
            .toList();
    }
}
C言語 (二重ループによるネストループ結合)

void register_field(DataDictionary* dict, const char* name, const char* type, const char* desc) {
    strncpy(dict->dictionary[dict->size].field_name, name, 50);
    strncpy(dict->dictionary[dict->size].data_type, type, 20);
    strncpy(dict->dictionary[dict->size].description, desc, 100);
    dict->size++;
}
